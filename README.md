# 🍽 Swiggy Clone Project 

This project is a frontend implementation of a food ordering application inspired by platforms like Swiggy.
It focuses on building a smooth user experience for browsing restaurants, exploring menus, and managing a cart.

---

## 🎥 Demo Video


![Demo Video](./demo/ProjectDemo.gif)


---

## 🚀 What this project does

The application allows users to:

* Explore a list of restaurants with basic details
* Open a restaurant to view its menu
* Browse items grouped in different sections
* Search for dishes
* Filter items (veg / non-veg)
* Add or remove items from cart
* View total bill on checkout page

---

## ✨ Highlights

* Clean and responsive UI
* Menu organized into collapsible sections
* Real-time cart updates
* Simple and intuitive navigation
* Handles API-based data rendering

---

## 🛠 Technologies Used

* React.js
* Redux Toolkit
* Tailwind CSS
* React Router
* Parcel (for bundling)

---

## 📦 How to run locally

1. Clone the repository

```bash
git clone https://github.com/your-username/swiggy-clone-frontend.git
```

2. Install dependencies

```bash
npm install
```

3. Start the project

```bash
npx parcel index.html
```
